##内容

这一单元主要是介绍一些基本的 python 语法知识：

* 数字 （1， 2， 3）
  * 运算符（+，-，\*，%，/）
  * 小数点（2.0/3）
* 字符串
  * 字符串连接（'a'+'b'，'a'\*\*2）
  * 字符串切分（'test'[:2]，'test'[:]，'test'[::-1]）
  * 字符串查找（'test'.find('t')，'tets'.find('t', 2)）
* 变量赋值
  * a = 1
  * b = a
  * c = b
  * a,b,c 的值是一样的

* str 函数
  * str(number) # 'number'

##收获
